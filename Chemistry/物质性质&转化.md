# 物质性质&转化

## 铁

|铁盐|亚铁盐|
|-|-|
|$\ce{Fe^{3+}}$|$\ce{Fe^{2+}}$|
|氧化性|氧化性&还原性|

铁盐氧化性：

- $\ce{2Fe^{3+} + Fe =\!=\!= 3Fe^{2+}}$
- $\ce{2Fe^{3+} + Cu =\!=\!= 2Fe^{2+} + Cu^{2+}}$
- $\ce{2Fe^{3+} + 2I^- =\!=\!= 2Fe^{2+} + I_2}$

亚铁盐氧化性：

- $\ce{Fe^{2+} + Zn =\!=\!= Zn^{2+} + Fe}$

亚铁盐还原性：

- $\ce{Fe^{2+} + Cl_2 =\!=\!= 2Fe^{3+} + Cl^-}$
